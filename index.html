Sobre MemoriasAprende más

Memorias pal diálogo es una aplicación con fines pedagógicos que busca ayudarte a hacer un diagnóstico de la forma en la que conceptualizas e interpretas el conflicto armado colombiano. La filosofía detrás de está aplicación sugiere que a mayor conocimiento propio, mayor curiosidad, y a mayor curiosidad, mayor capacidad de diálogo sobre asuntos colectivos fundamentales. 

Memorias pal diálogo ofrece una aplicación que puede ser usada por educadores o constructores de paz para facilitar conversaciones transformadoras a partir de la identificación de los modelos de memoria emblemática que individuos colombianos tienen sobre el conflicto. Esta aplicación se construye sobre trabajos académicos prestigiosos que discuten la memoria histórica en America Latina y en Colombia desde una perspectiva pluralista y decididamente orientada a la generación de diálogos improbables.  

Funciona en todas partes!

Mientras haya accesso a internet y a un dispositivo informático, Memorias pal diálogo puede utilizarse en cualquier parte para iniciar una conversación sobre nuestro pasado y el futuro al que debemos apuntar.

Se actualiza

Memorias pal diálogo es parte de un proyecto en humanidades digitales que se actualiza a partir de conversaciones académicas, nuevos hallazgos, y la respuesta de la comunidad de usuarios.

Código abierto

Crees que esta aplicación se puede mejorar, o que podrías utilizar el codigo base para hacer algo distinto? No hay problema! Esta aplicación se elaboró con un código abierto que seguirá abierto para todo público.
